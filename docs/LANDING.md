
# Sequoia DevTools

A Chrome extension that gives developers a dedicated DevTools panel and toolbar popup to inspect and manage feature flags in real time.

## Features

- **Inspect feature flags** — View all registered feature flags alongside their default and current values.
- **Toggle flags instantly** — Flip any boolean flag with a single click. Changes take effect immediately, no reload required.
- **Search and filter** — Quickly find flags by name using the built-in search bar.
- **Add custom flags** — Create new flags on the fly directly from the UI.
- **DevTools panel + popup** — Access flags from a full DevTools panel or a compact toolbar popup, whichever fits your workflow.
- **Auto-discovery** — The extension automatically detects feature flags on the page and keeps the display up to date.

## How It Works

Once installed, Sequoia DevTools adds a panel to Chrome DevTools and a popup accessible from the toolbar icon. Open either on a page running your application, and the extension will discover and display all active feature flags. Toggle, search, or add flags without touching code or reloading the page.

## Who Is This For?

Developers and QA engineers who work with feature flags and want a fast, visual way to inspect and override them during development and testing.

## Privacy

Sequoia DevTools does **not** collect, store, or transmit any user data. All feature flag information is read locally from the page and never leaves your browser. See the full [Privacy Policy](https://sokrati-manish.github.io/sequoia-devtools/PRIVACY) for details.
